# 🎬 Catálogo Animes

Esse projeto foi desenvolvido na 7ª Edição da Imersão Alura.

##  📚Sobre
Esse projeto é um catálogo de animes, no qual o usuário tem a total liberdade de inserir o anime de sua preferência juntamente com o banner.
